## Robot Platform

**Franka Emika Panda** — a 7-DOF collaborative robot arm with a parallel-jaw gripper, sourced from the [MuJoCo Menagerie](https://github.com/google-deepmind/mujoco_menagerie/tree/main/franka_emika_panda).

## Task Goal

A **long-horizon pick-and-place assembly task** where the Panda arm must:

1. Sequentially pick up **4 colored cubes** (red, green, blue, yellow) from a work table
2. Transport each cube to its **color-matched target zone**
3. Return to home position after each subtask

This requires planning and executing **~28 sequential phases** across 4 objects:
- **Per object:** Approach → Reach → Grasp → Lift → Transport → Place → Retreat
- **Total:** 28 phases + final home return

## Technical Approach

| Aspect | Implementation |
|--------|---------------|
| **Scene** | Custom MJCF with table, shelf, 4 floating cubes (free joints), 4 target zones |
| **Control** | Joint-space waypoint interpolation with smoothstep blending |
| **Gripper** | Position-controlled parallel jaw (0=closed, 0.04=open) |
| **Rendering** | Offscreen MuJoCo renderer → FFMPEG via imageio |
| **Camera** | Fixed orbit camera at 120° azimuth, -20° elevation |

## Core Features

- ✅ **Long-horizon planning**: 4 objects × 7 phases per object
- ✅ **Sequential execution**: Phase progression with smooth transitions
- ✅ **Grasp control**: Open/close gripper at precise timing
- ✅ **Physics simulation**: Free-joint cubes with full dynamics
- ✅ **Video output**: 1280×720 MP4 at 30 FPS

## Highlights

- Demonstrates the Panda's full reachable workspace
- Each cube has unique position offsets for varied trajectories
- Smoothstep interpolation prevents jerky robot motion
- Self-contained: single scene XML + single Python script

## Current Limitations

- Joint-space control (no inverse kinematics — waypoints are hand-tuned)
- No visual feedback / perception pipeline
- No collision avoidance (paths assume clear workspace)
